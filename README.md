# emacs-cheatsheet

Mini guide to Emacs keyboard shortcuts.

### Building

1. Install dependencies:

        sudo apt install git texlive

2. Grab the source:

        git clone https://github.com/00-matt/emacs-cheatsheet.git

3. Build the PDF:

        pdflatex emacs-cheatsheet.tex
